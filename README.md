# Portfolio Tehran IT

A lightweight, developer-focused WordPress portfolio plugin designed for custom themes and professional portfolio websites.

Portfolio Tehran IT provides a clean and flexible portfolio system based on WordPress Custom Post Types, Custom Taxonomies, reusable Template Parts, and custom project fields.

Unlike many portfolio plugins that focus on drag-and-drop builders, this plugin is designed primarily for developers who want complete control over structure, templates, frontend output, and theme integration.

---

## Key Features

- Lightweight architecture
- Portfolio Custom Post Type
- Portfolio Categories (Custom Taxonomy)
- Single Portfolio Pages
- Portfolio Archive Pages
- Portfolio Category Pages
- Portfolio Hero Sections
- Project Details & Features
- Image Galleries
- Video Support
- Download Links
- GitHub Repository Links
- Reusable Template Parts
- Global Portfolio Settings
- SEO-Friendly Structure
- Schema.org Markup
- Theme Integration Support

---

## Built For Developers

This plugin is intentionally built for developers rather than general users.

It assumes familiarity with:

- WordPress Development
- PHP
- Theme Development
- Template Customization
- Custom Post Types
- Custom Taxonomies

The plugin does not attempt to replace a page builder or visual website builder.

Instead, it provides a clean foundation that developers can integrate directly into their themes.
